# SHOPMATE

A minimalistic eCommerce store. Built with React/Redux. Features include product authentication, product browsing, cart, and a checkout flow with Stripe.

Enjoy.



## Getting Started

Run locally with 

```
npm run start
```


Build with 
```
npm run build
```


Built with 
```
React
Redux
Bootstrap 4
```
